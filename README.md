📝 Notes App (Java Console Application)

A simple Notes Management Application built in Java.
This app allows users to create, view, and delete notes, with all notes being saved to a text file (notes.txt) for persistence.


---

🚀 Features

Add new notes

View all saved notes

Delete a specific note

Notes are stored in notes.txt (persistent storage)

User-friendly console menu



---

📂 Project Structure

Notes.java      // Main source code
notes.txt       // File where notes are stored
README.md       // Project documentation


---

🛠️ Requirements

Java Development Kit (JDK 11 or later recommended)

Any Java IDE (IntelliJ IDEA, VS Code, Eclipse) or terminal to run



---

▶️ How to Run

1. Clone or Download the project


2. Open a terminal in the project folder


3. Compile the code:

javac Notes.java


4. Run the program:

java Notes


5. Follow the on-screen menu to manage notes

🗂️ File Handling

Notes are saved in notes.txt automatically

On exiting, all notes are updated and stored

If notes.txt does not exist, it will be created automatically
